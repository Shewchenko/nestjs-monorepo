## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
#app
$ sudo apt-get install apache2-utils
$ npm run start:dev
$ nest start sub-project --watch
#run
$ ab -n 20 -c 10  http://localhost:3000/1

# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
